Page d'aide

Sur cette page vous trouverez les différentes fonctions de cette application.

"Add a user" qui permet d'ajouter une personne dans la base de donnée en remplissant les champs suivant:

  • Nom
  • Prenom
  • Sexe
  • Métiers
  • Service
  • Adresse Mail
  • Telephone
  • Résumé
Après avoir créé un "user" on a la possibilité de cliquer dessus afin d'accéder à ses informations.
Quatre boutons se présentent à vous:
  • Delete
  • Modify
  • Call
  • Send Mail
Comme leurs noms l'indique, on peut "supprimer", "modifier", "appeler" ou encore "envoyer un mail".

Voilà les principales fonctions de l'application, en attendant d'autres fonctionnalités.
