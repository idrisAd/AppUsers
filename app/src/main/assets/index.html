Page d'aide

Sur cette page vous trouverez les différentes fonctions de cette applications.
Tout d'abord vous avez un bouton "Add a user" qui permet d'ajouter une personne dans la base de donnée en remplissant certains champ:

  • Nom
  • Prenom
  • Sexe
  • Métiers
  • Service
  • Adresse Mail
  • Telephone
  • Résumé
